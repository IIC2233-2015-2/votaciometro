Votaciómetro API Privada

Para usar esta API necesitas las credenciales suficientes para realizar HTTP Basic Auth.

Se recomienda leer la documentación de la librería que uses para saber cómo realizar esta operación.

Uso

GET /api/v1/lists

Retorna una lista en formato JSON con todas los nombres de las listas presentes en la elección. Por ejemplo: 

[
  "list_1",
  "list_2",
  "list_3"
]

GET /api/v1/tables

Retorna una lista en formato JSON con todas las mesas disponibles. Por ejemplo: 

[
  {
    "_id": "563abbc56bb02c49ca701bea",
    "name": "Table"
  },
  {
    "_id": "563abbc56bb02c49ca701bec",
    "name": "Other Table"
  }
]

GET /api/v1/tables/:id

Retorna los resultados de esa mesa en formato JSON. Por ejemplo: 

{
  "_id": "563abbc56bb02c49ca701bea",
  "name": "Table",
  "votes": {
    list_1: 104,
    list_3: 12,
    list_2: 132
  }
}
