Votaciómetro API Privada

Para usar esta API necesitas las credenciales suficientes para realizar HTTP Basic Auth.

Al realizar una operación sin credenciales o con credenciales inválidas se retornará Status Code 401 Unauthorized.

Se recomienda leer la documentación de la librería que uses para saber cómo realizar operaciones usando este protocolo.

Uso

Toda acción retorna un Status Code 200 OK a menos que se explicite lo contrario.

GET /api/v1/lists

Retorna una lista en formato JSON con todas los nombres de las listas presentes en la elección. Por ejemplo: 

[
  "list_1",
  "list_2",
  "list_3"
]

GET /api/v1/tables

Retorna una lista en formato JSON con todas las mesas disponibles. Por ejemplo: 

[
  {
    "_id": "563abbc56bb02c49ca701bea",
    "name": "Table"
  },
  {
    "_id": "563abbc56bb02c49ca701bec",
    "name": "Other Table"
  }
]

GET /api/v1/tables/:id

Retorna los resultados de los votos de una mesa en formato JSON, se debe identificar a la mesa usando su id.

Por ejemplo, GET a /api/v1/tables/563abbc56bb02c49ca701bea retorna: 

{
  "_id": "563abbc56bb02c49ca701bea",
  "name": "Table",
  "votes": {
    list_1: 104,
    list_3: 12,
    list_2: 132
  }
}
